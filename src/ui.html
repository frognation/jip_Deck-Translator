What is jip Translator?

A Figma plugin that translates text across selected frames and pages using AI. Works with Figma Design, FigJam, and Figma Slides.

Key Features

  • Context-aware translation — Reads all text in order across selected frames/pages, understands the full context, then translates coherently.
  • Multiple AI models — Supports OpenAI GPT, Anthropic Claude, Google Gemini, and DeepL.
  • Flexible output — Replace original, append translation, or duplicate with auto-layout.
  • Custom guidelines — Use preset templates or write your own translation instructions. Upload .md files for detailed guidelines.
  • Auto-detect source language — Or manually specify source and target languages.

How to Use

  1. Select one or more frames/pages in Figma
  2. Choose your AI model and enter API key if needed
  3. Set source and target languages
  4. Choose output mode (replace / append / duplicate)
  5. Optionally add translation guidelines
  6. Click Translate

Tips

  • Select multiple frames to translate them in context together.
  • Use "Slide / Presentation" template for deck translations.
  • API keys are saved locally in your browser and never sent to any server except the AI provider.

Made with care for designers who work across languages.
